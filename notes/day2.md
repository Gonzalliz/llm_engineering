## LLM Engineering

### Frontier models
1. Closed-Source:
    - GPT-4 (OpenAI)
    - Claude (Anthropic)
    - Gemini (Google DeepMind)
    - PaLM (Google)


2. Open-Source:
    - LLaMA (Meta)

### Three ways to use models

1. Using Chat interfaces
    - Ex: ChatGPT
2. Cloud APIs
    - Ex: LLM API, Frameworks like langchain
    - Manages Ai cloud services
        - Amazon bedrock
        - Google vertext 
        - Azure ML
3. Direct inference
    - With HugginFace transformer library
    - With Ollama to run locally
